A C# console-based application designed for Salisbury University to introduce a new IQ testing system for MBA students. The program evaluates students across multiple subjects and calculates their IQ level based on their performance.

ğŸ¯ Project Purpose
Salisbury University wanted to enhance its traditional examination system by adding an IQ testing module. This application allows students to:

Enter their name and ID

Answer 5 multiple-choice questions across Aptitude, English, Mathematics, and General Knowledge

Receive a score and bonus points based on performance

View their IQ level classification

Choose to continue or exit after each test

This project demonstrates how to build interactive, menu-driven applications in C# using classes, loops, and conditional logic.
